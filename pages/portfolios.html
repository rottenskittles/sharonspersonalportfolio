just in case you were curious, here are some examples of the things that i like to do

web/programming

This is my first website I created, a dinosaur dossier. I made it in my Web Applications class at CART using HTML and CSS in Visual Studio Code. While I was making it, I definitely faced a lot of problems and challenges, but because I was able to reach out whenever I needed help on something, I was able to finish it successfully. Although it isn’t the best, I am still proud of it.

This is my latest website that me and a partner made before this one, also made using HTML and CSS along with JavaScript in Visual Studio Code. We were assigned to make a text adventure-like game with various endings. This project was definitely difficult, because we don’t usually make things like this, but with the help of my teacher and some of my friends at CART, we were able to make a functioning game and I’m happy it turned out well.

writing

This is a short story I wrote in early May of 2024 using Google Docs. Coming up with certain parts of the story was difficult; I would run into writer blocks a lot of the time. I was able to get over it by taking some time off my work and allowing ideas to flow on their own before coming back to it. This taught me that even though working hard is important, taking time for myself is important as well.
